# **App Name**: Cricket Oracle

## Core Features:

- User Authentication: Allow users to create accounts and log in using email authentication.
- API Integration: Fetch cricket series data, including game schedules and team information, from the Cricapi API (https://api.cricapi.com/v1/series_info?apikey=6cbb4198-8e11-46e3-8dd5-f353458e68c1&id=0cdf6736-ad9b-4e95-a647-5ee3a99c5510).
- Game Prediction Input: Enable users to submit their predictions for upcoming cricket games.
- Prediction Storage: Store user predictions in a database.
- Prediction Evaluation: Automatically evaluate the accuracy of user predictions based on actual game outcomes fetched from the API.
- Leaderboard: Display a leaderboard ranking users based on their prediction accuracy.
- Game Outcome Forecast Tool: Employ a generative AI LLM tool, using player statistics and match conditions, to decide on a winning percentage for each participating team. This data will enhance users' prediction process by providing insightful AI-driven recommendations for potential outcomes.

## Style Guidelines:

- Primary color: Dark teal (#008080) to represent the strategic nature of cricket.
- Background color: Off-white (#F0F8FF) for a clean, professional look.
- Accent color: Gold (#FFD700) to highlight top performers and important data.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look suitable for headlines or body text.
- Use cricket-themed icons (bats, balls, wickets) for game-related actions and data visualization.
- Organize information with a clear, grid-based layout, emphasizing game schedules, user predictions, and leaderboard standings.
- Use subtle animations to indicate changes in the leaderboard or new game updates.
